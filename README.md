# ANSC 595 Microbiome Project

This repository contains the scripts used for QIIME2 processing and downstream R analyses for the final microbiome manuscript.

## Repository contents
- QIIME2 processing scripts
- DESeq2 differential abundance analysis
- Random Forest classification
- Co-occurrence network analysis
- Figure generation scripts

## Main metadata variable
- treatment

## Main analyses
- alpha diversity
- beta diversity
- taxa bar plots
- ANCOM-BC differential abundance
- DESeq2 differential abundance
- random forest
- co-occurrence
- DADA2 denoising statistics
